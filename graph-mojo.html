depgraph:graph

Full name:

com.github.ferstl:depgraph-maven-plugin:1.0.1:graph

Description:

Creates a dependency graph of a maven module.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency collection of artifacts in scope: test.
  • The goal is thread-safe and supports parallel builds.

Optional Parameters

Name Type Since Description
createImage boolean 1.0.0 If set to true and Graphviz is installed on the system where this plugin is executed, the dot file will be converted to a graph image using Graphviz' dot executable.
Default value is: false.
User property is: createImage.
dotExecutable File 1.0.0 Path to the dot executable. Use this option in case createImage is set to true and the dot executable is not on the system PATH.
User property is: dotExecutable.
excludes List 1.0.0 Comma-separated list of artifacts to be excluded in the form of groupId:artifactId:type:classifier.
User property is: excludes.
imageFormat String 1.0.0 The format for the graph image when createImage is set to true.
Default value is: png.
User property is: imageFormat.
includes List 1.0.0 Comma-separated list of artifacts to be included in the form of groupId:artifactId:type:classifier.
User property is: includes.
outputFile File 1.0.0 The path to the generated dot file.
Default value is: ${project.build.directory}/dependency-graph.dot.
User property is: outputFile.
scope String 1.0.0 The scope of the artifacts that should be included in the graph.
User property is: scope.
showConflicts boolean 1.0.0 If set to true, the graph will additionally contain conflicting dependencies. Note that the dependency graph may not be 100% accurate when this flag is enabled and the plugin is executed with a Maven version greater or equal 3.0!
Default value is: false.
User property is: showConflicts.
showDuplicates boolean 1.0.0 If set to true, the graph will additionally contain duplicate dependencies. Note that the dependency graph may not be 100% accurate when this flag is enabled and the plugin is executed with a Maven version greater or equal 3.0!
Default value is: false.
User property is: showDuplicates.
showVersions boolean - If set to true, the created graph will show version information an all artifacts. Depending on the flags showVersions and showConflicts, the version will either be shown directly in the artifact nodes or on the dependency edges.
Default value is: false.
User property is: showVersions.
